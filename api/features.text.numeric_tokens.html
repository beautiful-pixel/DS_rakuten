features.text.numeric_tokens.replacer.replace_numeric(match, measure_type, unit_pos=None, int_pos=None)[source]

Remplace une expression numérique détectée par un token sémantique.

Cette fonction : - extrait les valeurs numériques capturées par une expression régulière, - normalise et convertit les unités si nécessaire, - discrétise la valeur obtenue selon le type de mesure, - retourne un ou plusieurs tokens textuels.

Paramètres:

  • match (re.Match) – Correspondance regex contenant les groupes capturés.

  • measure_type (str) – Type de mesure (ex: “volume”, “length”, “age”).

  • unit_pos (int, optional) – Position du groupe correspondant à l’unité.

  • int_pos (int, optional) – Position du premier élément d’un intervalle.

Renvoie:

Token(s) textuel(s) remplaçant l’expression numérique.

Type renvoyé:

str

class features.text.numeric_tokens.transformers.NumericTokensTransformer(designation_col='designation', description_col='description')[source]

Bases : BaseEstimator, TransformerMixin

Transformateur scikit-learn remplaçant les expressions numériques dans le texte par des tokens sémantiques.

Ce transformateur applique une tokenisation numérique sur les champs textuels (titre et description) en remplaçant les valeurs quantitatives (dimensions, poids, dates, quantités, etc.) par des tokens discrets.
